# Contributing to Speedify2

Thank you for your interest in contributing! Every contribution helps make Speedify2 better.

## Ways to Contribute

- Fix bugs
- Improve UI
- Add features
- Improve documentation
- Improve performance

## Development Setup

```bash
# Clone the repository
git clone https://github.com/bnp-aswin/speedify2
cd speedify2

# Install dependencies
pip install -r requirements.txt

# Run the app
python speedify.py
```

## Pull Request Process

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/my-feature`)
3. Commit your changes (`git commit -m 'Add my feature'`)
4. Push your branch (`git push origin feature/my-feature`)
5. Open a Pull Request

Please ensure your code is clean and well documented.

## Good First Issues

New to the project? Look for issues labeled **`good first issue`** — these are approachable tasks for first-time contributors:

- Add Linux support
- Add macOS support
- Add system tray support
- Add mini speed graph
- Add theme switching
- Improve UI customization

## Code Style

- Keep it simple and readable
- Follow existing code patterns
- Avoid unnecessary dependencies

## Questions?

Open an issue and tag it with `question`. We're happy to help!
